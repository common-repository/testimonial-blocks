 Easy to use block for adding testimonials to pages.

== Description ==
 The plugin adds a testimonial block type.

== Installation ==
  Upload the Testimonial Block to your WordPress blog and activate it.
  Go to 'Add New' under Posts in the Dashboard.
  Click on (+) 'Add block'
  In the list of categories, click on 'Common Blocks'
  Choose the Testimony block type and begin typing.
